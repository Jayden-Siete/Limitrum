# @limitrum/db

Local SQLite/libSQL policy and audit store for Limitrum.

This package provides the schema, local client, migration bootstrap, and seed data used by the open-source policy kernel.

```bash
pnpm add @limitrum/db
```

```bash
pnpm --filter @limitrum/db migrate
pnpm --filter @limitrum/db seed
```

See the main repository README for the full MVP workflow.
