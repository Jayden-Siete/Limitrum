Capabilities

Every guard your CISO
has been asking for.

Domain Allowlisting
Restrict your agent to an exact list of external domains. Any call to an unlisted endpoint is silently dropped and logged with full payload.
Budget Caps
Set hard daily, weekly, or per-action financial limits. The kernel rejects any action that would breach your configured budget — before it hits the API.
Loop Detection
Automatic runaway protection. If an agent triggers N identical actions within T seconds, Limitrum kills the cycle and fires a configurable alert immediately.
Syscall Protection
Block any attempt to spawn processes, execute shell commands, or access the filesystem outside defined boundaries — regardless of the prompt.
Full Audit Trail
Every action is logged with timestamp, agent ID, payload hash, policy version, enforcement decision, and reason. Immutable, queryable, exportable.
Out-of-Band Kernel
The Policy Kernel runs in a completely separate runtime. A jailbroken system prompt cannot override it. The agent has no knowledge of the kernel's existence.
